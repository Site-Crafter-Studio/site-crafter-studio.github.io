References


A list of references used during the creation of Site Crafter Studio. These websites are also useful tools in helping you to expand and develop your coding knowledge.



    • CSS Gradient

    CSS Gradient lets you create free gradient backgrounds for your website. Besides being a gradient generator, the site is also contains content about gradients from technical articles to real life gradient examples, like Stripe and Instagram.



    • HTML Arrows by Toptal Designers

    HTML Arrows is a comprehensive reference website for finding HTML symbol codes and entities, ASCII characters and Unicode hexadecimal values to use in your web design.



    • HTML Color Codes

    Get HTML color codes, Hex color codes, RGB and HSL values with the color picker, color chart and HTML color names.



    • Font Awesome

    Font Awesome is the Internet's icon library and toolkit, used by millions of designers, developers, and content creators.



    • freeCodeCamp

    freeCodeCamp is a non-profit organization that consists of an interactive learning web platform, an online community forum, chat rooms, online publications and local organizations that intend to make learning software development accessible to anyone.



    • W3Schools

    W3Schools is a freem educational website for learning coding online. It offers courses covering many aspects of web development, from HTML and CSS, to JavaScript and Python.



    • Unsplash

    Unsplash is a photo discovery platform for free to use, high-definition photos. Images are uploaded by photographers from around the world.



    • World Wide Web Consortium (W3C)

    W3C develops standards and guidelines to help everyone build websites based on the principles of accessibility, privacy, and security.
